Search bar

In the last submission I fetched the different categories from the WordPress API to filter content. It's always useful to have the option to search, so I decided to try to make one.

I had no idea how to go about it, so I used this tutorial to get an idea on how to do it. With the help of this I was able to make the searchbar functional using the keyup event to see if the search string matches any of the posts title, excerpt or content.

As of now results come up as you type.
