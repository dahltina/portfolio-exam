Changes

The feedback I got from this submission was mainly some technical issues, amongst them a horizontal scrollbar on the exhibitions page, that I had too long paragraphs as well as some div's that didn't get closed.

I fixed these issues, and made some other changes and improvements.

Contact page

The contact page consists solely of a form. In the initial submission I hadn't put much work into styling it, so there was potential for improvements.

I hadn't set a max-width on the form on tablet sized screens, so I set it to 800px in order for it not to become too wide.

I added some styling like padding, a black border to all the input fields and set the font-family and font-size to be the same on all inputs.

Added client side form validation.

Other minor improvements

  • Added transition to image hover effect and menu hover
  • The layout on the index page on mobile size was a bit crammed, so I added padding here and there, for example between the cta boxes.
  • On mobile and tablet sized screens, the active pages in the menu didn’t have a color to differentiate from the other sites, so I set it to red as on laptop-sized screens.
  • Linked to specific parts of pages. For example, the «Explore robotics and AI» CTA on the index page now links to that section on the exhibitons page. I also linked up all the footer items to the correct parts of the pages.
